# Changelog v1.1.7

### Added
- Added `Timer.add()`, `Timer.set()`, `Timer.isOver()`
    - Example:
    ```js
    Timer.add(timer, runOnce, ()=>{tellraw @s "POG";}, @a);

    execute as @a run {
        Timer.set(timer, @s, 100);
    };

    Trigger.setup(test, {
        1: ()=>{
            if (Timer.isOver(timer)) {
                tellraw @s "over";
                Timer.set(timer, @s, 100);
            } else {
                tellraw @s "not_over";
            }
        }
    })
    ```
