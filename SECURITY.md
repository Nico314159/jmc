# Security Policy

## Supported Versions

Try to avoid using older versions of JMC. No security vulnerability has been found so far.
| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

Create a github issue and ping me in discord if it's severe.
